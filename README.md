# ChirpCircle

ChirpCircle is a simple social network written with [MERN](http://mern.io).

## Quickstart

```
  git clone <uri>
  cd chirpcircle
  npm install
  <launch mongo>
  npm start
```

## Deployment

A `Dockerfile` is included which starts a nodejs process running the application.
